PROMPT REFINEMENT SUGGESTIONS

Based on this session, how can we improve our prompts?

1. WORKING AGREEMENTS REFINEMENT
   - Were any agreements unclear or insufficient?
   - Should we add new quality gates?
   - Do any constraints need adjustment?

2. ANALYSIS PROMPT IMPROVEMENT
   - What information was missing from initial analysis?
   - What questions should we always ask up front?
   - What searches or investigations should be standard?

3. PLANNING PROMPT ENHANCEMENT
   - Was the plan detailed enough?
   - What planning elements were most valuable?
   - What was missing that caused issues later?

4. IMPLEMENTATION PROMPT OPTIMIZATION
   - Which TDD rules were most important?
   - What guardrails prevented mistakes?
   - What additional constraints would help?

5. CHECK PROMPT ADDITIONS
   - What quality aspects did we miss checking?
   - What verification steps would catch more issues?
   - What documentation was inadequate?

Provide 2-3 specific, actionable prompt improvements.
